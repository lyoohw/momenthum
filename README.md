# momentum 모멘텀
현재시각을 확인할 수 있고 To Do List를 작성할 수 있으며, 지금 나의 위치와 날씨를 알 수 있는 프로그램입니다!
- <https://lyoohw.github.io/momentum/>
 
 
## 개요
- 프로젝트 이름: momentum
- 작업 기간: 2023.06.01 ~ 2023.06.16
- 사용한 스킬: JavaScript, LocalStorage
- 개인 프로젝트


## 구현기능
- 배경이미지 랜덤으로 변경
- 로그인
- 현재 시각 표시
- 현재 날짜, 요일 표시
- To Do List 작성
- To Do List 체크시 취소선
- To Do List 삭제
- 명언 랜덤 표시
- 나의 지역, 날씨, 기온 표시(API사용)


  ## 프로젝트 화면
  - 로그인 페이지
